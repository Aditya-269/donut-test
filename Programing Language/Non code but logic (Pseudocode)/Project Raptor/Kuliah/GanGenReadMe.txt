GanGen.exe adalah sebuah program untuk mencari ganjil atau genap dalam range angka 0 hingga x dimana x adalah batas maksimum yang ditentukan oleh user.

aplikasi ini menggunakan prinsip modulo sesuai dengan materi minggu ini yaitu modulo atau mod (%)
modulo di aplikasi berperan penting dalam menentukan apakh bilanga tersebut ganjil atau genap.
dengan rumus dasar x%2=0 adalah bilangan genap.

Fitur :
1. memilih apyang ingin dicari
2. menginput batasan sesukanya
